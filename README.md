# ATM System

## Overview

The ATM System is a simple Python application built using Object-Oriented Programming (OOP). It simulates basic ATM operations such as depositing money, withdrawing money, checking the account balance, and displaying account details.

## Features

* Create an ATM account
* Deposit money
* Withdraw money with insufficient balance validation
* Check available balance
* View account holder details

## Technologies Used

* Python 3
* Object-Oriented Programming (OOP)

## How to Run

1. Save the program as `atm_system.py`.
2. Open a terminal or command prompt.
3. Navigate to the project folder.
4. Run the program:

```bash
python atm_system.py
```

## Sample Output
